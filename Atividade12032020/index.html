Tags HTML:

Nome da tag Função
!DOCTYPE html Informa ao navegador o site que está sendo utilizado.
html Contém todos os elementos de um documento.
head São as tags HTML.
title Define o título da sua página, o nome que aparecerá na sua aba, janela ou guia.
body É nessa tag que os elementos visíveis do seu site devem ser inseridos.
p Faz um parágrafo.
strong Transforma o conteúdo em negrito.
i Transforma o conteúdo em itálico.
br Essa tag não necessita de fechamento, ela executa a função de quebra de linha.
hr Essa tag não necessita de fechamento, ela forma uma linha horizontal.
tr Forma linhas em uma tabela.
td Forma uma tabela.
h1,h2,h3,h4,h5 Serve para aumentar o título da página.

Atributos das Tags HTML:

Nome do atributo Função do atributo
id É uma forma de identificar um elemento, e devem ser únicas para cada elemento. É como se fossem impressões digitais de nossos dedos ou RGs.
