Nome da Tag Função
!DOCTYPE html Informa ao navegador a versão do html que esta sendo utilizado.
html Contêm todos os elementos de seu documento, todo o documento html deve ser inicíado e finalizado com essa tag.
title Define o título da sua página, o nome que aparecerá na sua aba, janela ou guia.
body É nessa tag que todos os elementos visíveis do seu site devem ser inseridos, pois ela representa o corpo do documento.
p Forma um parágrafo.
i Transforma o conteúdo em itálico.
hr Essa tag não necessita de fechamento, ela forma uma linha horizontal.
strong Tranforma o conteúdo em negrito.
tr Forma a linha de uma tabela em html.
td Forma a célula de uma tabela em html.

Explicando as seguintes atributos:

Nome do atributo Função do atributo
id É uma forma de identificar um elemento, e devem ser únicas para cada elemento. É como se fossem impressões digitais de nossos dedos ou RGs. Através delas, pode-se atribuir formatação a um elemento em especial.
style Permite incluir elementos CSS (estilos).
class Uma forma de identificar um grupo de elementos. Através dele, pode-se atribuir formatação a vários elementos de uma vez.
